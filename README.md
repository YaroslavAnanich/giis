# Визуализатор алгоритмов построения отрезков

## Цель проекта
Создание интерактивного образовательного инструмента для:
- Сравнительного анализа алгоритмов растеризации
- Визуализации математических принципов построения линий
- Демонстрации особенностей работы с дискретными структурами

## Основные задачи
1. Реализация трех классических алгоритмов:
   - Цифровой дифференциальный анализатор (CDA)
   - Алгоритм Брезенхема (целочисленная версия)
   - Алгоритм Ву с антиалиасингом
2. Разработка интуитивного интерфейса:
   - Интерактивный ввод параметров
   - Визуальная обратная связь
   - Адаптивная система координат
3. Создание режима отладки:
   - Пошаговая анимация
   - Подсветка текущей операции
   - Визуализация процесса принятия решений

## Полный набор функций

### Основные возможности
| Функция                  | Описание                                  |
|--------------------------|-------------------------------------------|
| Выбор алгоритма          | 3 метода построения из выпадающего списка |
| Ручной ввод координат    | 4 числовых поля для X1,Y1 и X2,Y2         |
| Автоматическая сетка     | 15x15 пикселей с серой подложкой          |
| Режим отладки            | Активация пошагового выполнения           |
| Анимация построения      | Визуализация с задержкой 50 мс            |
| Сброс состояния          | Очистка canvas при изменении параметров   |

### Особенности реализации
- Динамическая система координат (лево-верхний угол = (0,0))
- Адаптивная перерисовка при изменении размеров окна
- Сохранение исходных настроек контекста canvas
- Оптимизированная работа с пикселями

## Подробная инструкция по использованию

- Заполните все четыре поля:
  - X1, Y1 - координаты начала отрезка
  - X2, Y2 - координаты конца отрезка
- Пример корректных значений:
  ```text
  X1: 3, Y1: 2
