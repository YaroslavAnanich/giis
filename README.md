# Приложение для визуализации алгоритмов построения линий

Веб-приложение для отрисовки отрезков с использованием различных алгоритмов с возможностью пошаговой отладки.

## Особенности

- Реализация трех алгоритмов:
  - Цифровой дифференциальный анализатор (CDA)
  - Алгоритм Брезенхема
  - Алгоритм Ву (Xiaolin Wu)
- Режим отладки с визуализацией шагов
- Анимированное построение линии
- Сетка с шагом 15px для наглядности
- Подсветка текущего блока в режиме отладки
- Отображение порядкового номера шага в блоке

## Как использовать

1. Введите координаты начала (x1, y1) и конца (x2, y2) отрезка
2. Выберите алгоритм построения из выпадающего списка
3. При необходимости включите режим отладки
4. Нажмите кнопку "Нарисовать отрезок"

В режиме отладки:
- Каждый шаг алгоритма выделяется цветом
- Отображается порядковый номер шага
- Используется задержка 50 мс между шагами
- Автоматически отрисовывается сетка

## Требования

- Современный браузер с поддержкой:
  - HTML5 Canvas
  - ES6 классов
  - async/await
